# E-Store Website - Frontend

A modern, responsive e-commerce web application built with **React** and **Tailwind CSS**.

## Features

- **Responsive Design** - Mobile-first approach with seamless experience across all devices
- **Product Catalog** - Browse and search through product listings
- **Shopping Cart** - Add/remove items with real-time updates
- **Product Filtering** - Allows filtering by category (price and rating filters are not available yet)
- **Checkout Process** - Smooth and intuitive checkout experience
- **Product Details** - Comprehensive product information and reviews
- **Wishlist** - Save favorite items for later
- **Login** - User authentication page
- **Shipping** - Shipping information page
- **Order Review** - Review order details before placing it
- **Success Order** - Order confirmation page
- **Contact Us** – Contact form page for customer inquiries and support
- **Responsive Navigation** - Easy-to-use navigation bar and menu

## Tech Stack

- **React Js** - UI library for building dynamic interfaces
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **React Router** - Client-side routing and navigation
- **State Management** - Context API
